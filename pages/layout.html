Estrutura de Layout

Nesta fase defines como a informação se organiza no espaço.
Não escolhes cores, não defines marca, não fazes design final. O layout é uma resposta direta à estrutura que já foi decidida.

💡 Pensa no layout como um tabuleiro. Aqui decides onde as coisas vivem, não como parecem.

Tudo o que definires aqui vai servir de base para o Branding, Acessibilidade e SEO.

Grelha base do layout

A grelha define os limites do conteúdo e garante consistência em todas as páginas do projeto.

Layout das páginas

Agora aplica a grelha às páginas que definiste existir.
Não é design final — é estrutura base.

✅ Cada página usa a mesma grelha, mas pode organizar o conteúdo de forma diferente.

ℹ️ O layout é definido página a página. Escolhe uma página.

Componentes globais

Componentes globais são partes do layout que se repetem em várias páginas para criar estabilidade.

Estes componentes normalmente refletem decisões feitas na Estrutura Base.

Hierarquia visual

Aqui defines o que deve chamar atenção primeiro e o que tem menor prioridade visual.
